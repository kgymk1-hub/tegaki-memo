# tegaki-memo（手書きメモ）

ブラウザだけで動く、シンプルな手書きメモ・お絵かきアプリです。HTML / CSS / JavaScript の静的ファイルだけで構成しているため、GitHub Pages で公開できます。スマホ・タブレット・PC から、指・タッチペン・マウスでメモやラフスケッチを作成できます。

## GitHub Pages URL

```text
https://kgymk1-hub.github.io/tegaki-memo/
```

## 主な機能

- canvas への手書き描画
- ペン / 消しゴム
- 色変更（プリセット色・任意色）
- 太さ変更
- Undo
  - 描画
  - 現在レイヤーの全消去
  - 画像読込
  - レイヤー追加 / 削除 / 名前変更 / 表示・非表示 / 順序変更 / 下へ結合
  - 背景モード変更 / 背景色変更
  - レイヤー透明度変更
- 背景設定
  - 白
  - 任意色
  - 透明（画面上はチェック柄）
- 画像読込
  - 新規レイヤーとして読み込み
  - 現在レイヤーへ読み込み
- PNG保存
  - 表示中レイヤーのみを書き出し
  - 背景設定とレイヤー透明度を反映
- レイヤー管理
  - 最大5枚
  - 追加 / 削除
  - レイヤー名変更
  - 表示 / 非表示
  - 上下移動
  - 下へ結合
  - 透明度調整
- PWA対応
  - `manifest.json`
  - `service-worker.js`
  - ホーム画面追加対応
  - アプリシェルのキャッシュ対応

## 使い方

1. GitHub Pages URL、またはローカルの `index.html` をブラウザで開きます。
2. キャンバス上を指・タッチペン・マウスでなぞると描画できます。
3. ツールバーでペン、消しゴム、色、太さ、背景、レイヤーを切り替えます。
4. 「読込先」で画像を「新規レイヤー」または「現在レイヤー」に読み込むか選び、「画像」ボタンから画像ファイルを選択します。
5. レイヤーの「透明度」スライダーで選択中レイヤーの見え方を調整できます。
6. 「戻す」で直前の操作をUndoできます。
7. 「PNG保存」で現在の表示内容を画像として保存できます。

## ファイル構成

```text
tegaki-memo/
├─ index.html         # 画面構造、PWA manifest / service worker 登録
├─ style.css          # レイアウト・見た目・スマホ向け調整
├─ app.js             # 描画処理・Undo・背景・画像読込・レイヤー管理・PNG保存
├─ manifest.json      # PWA設定
├─ service-worker.js  # キャッシュ制御
└─ icons/
   └─ icon.svg        # PWAアイコン
```

## 実装方針メモ

- 既存の操作感を大きく変えず、ツールバーは機能ごとに行を分けています。
- Undo履歴にはレイヤー状態、アクティブレイヤー、ヒント表示状態、背景設定を保存しています。
- 下へ結合は非表示レイヤーが絡む場合に禁止し、見た目と結果がずれないようにしています。
- 画像読込は「新規レイヤー」を初期値にしつつ、「現在レイヤー」へ直接読み込む従来仕様も残しています。
