# tegaki-memo（手書きメモ）

ブラウザだけで動く、シンプルな手書きメモ・お絵かきアプリです。スマホ・タブレット・PCで、指・タッチペン・マウスを使ってメモやラフスケッチを作成できます。

## デモURL

GitHub Pagesで公開する場合は、以下のようなURLになります。

```text
https://<ユーザー名>.github.io/tegaki-memo/
```

> 公開後に実際のURLへ差し替えてください。

## 主な機能

- **手書き描画**
  - Pointer Eventsによる指・タッチペン・マウス対応
  - 滑らかに見える曲線補間
  - 高DPI環境対応（描画負荷を抑えるため倍率は最大1.5）
- **ペン**
  - 黒・赤・青・緑・任意色
  - 細い・普通・太い・極太の太さ選択
- **消しゴム**
  - 選択中レイヤーの描画だけを消去
  - 背景や他レイヤーには影響しません
- **Undo**
  - 最大10件まで履歴を保持
  - 描画、現在レイヤーの全消去、画像読込、レイヤー削除、順序変更、結合、名前変更に対応
- **背景設定**
  - 白背景
  - 任意色背景
  - 透明背景（画面上はチェック柄表示）
- **画像読込**
  - 画像ファイルを現在のレイヤーへ読み込み
  - キャンバス内に収まるように縮小し、中央へ配置
- **PNG保存**
  - 表示中レイヤーのみを書き出し
  - 背景設定に応じて白背景・色背景・透明PNGを保存
- **レイヤー管理**
  - 最大5枚
  - 追加、削除、表示／非表示、上下移動、下へ結合、名前変更

## 使い方

1. `index.html` をブラウザで開きます。
2. キャンバス上を指・タッチペン・マウスでなぞると描画できます。
3. ツールバーでペン、消しゴム、色、太さ、背景、レイヤーを切り替えます。
4. 「画像読込」から画像を現在のレイヤーへ配置できます。
5. 「PNG保存」で現在の表示内容を画像として保存できます。

## ファイル構成

```text
tegaki-memo/
├─ index.html  # 画面構造
├─ style.css   # レイアウト・見た目
└─ app.js      # 描画処理・Undo・背景・画像読込・レイヤー管理・PNG保存
```

## 現在の注意点

- 「全消去」ボタンは、アプリ全体ではなく**現在選択中のレイヤーだけ**を消去します。
- レイヤー追加、表示／非表示、背景変更は、現時点ではUndo対象外です。
- 画像読込は現在のレイヤーへ直接描画されます。読み込み後の移動・拡大縮小・回転は未対応です。
- サーバー処理はなく、静的ファイルだけで動作します。
